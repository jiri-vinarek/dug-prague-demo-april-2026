# Notebook-Based Development: From EDA to ETL Pipelines

Presented at **Databricks User Group Prague**, April 2026.

## About

This project demonstrates how notebook-based development can take you from exploratory data analysis all the way to production ETL pipelines. It showcases **literate programming** — weaving narrative, code, and results into a single document — and how [nbdev](https://nbdev.fast.ai/) can be used to export notebook code into reusable Python modules.

## Dataset

The use case is demonstrated on the [Prague Institute of Planning and Development (IPR) open data](https://pid.cz/en/opendata/) dataset. The data is licensed under [CC-BY](https://creativecommons.org/licenses/by/4.0/).

## What You'll Find

* **Exploratory Data Analysis** — interactive notebooks for understanding and visualizing data
* **ETL Pipelines** — notebooks that evolve from exploration into structured data transformations
* **Code Export with nbdev** — examples of extracting tested, documented functions from notebooks into importable modules
